# HTML→XLSX конвертер (экспорт 1С) — standalone

Локальное приложение (Electron) для конвертации одного или нескольких HTML-файлов (типичный экспорт 1С) в XLSX с сохранением форматирования: границы, заливки, шрифты, выравнивания, объединения ячеек, размеры строк/колонок.

Требования и пример входного HTML: `ТЗ.html`, `Тест макета.html`.

## Зависимости

- **Node.js** 18+
- npm (идёт с Node.js)

## Установка и запуск

```bash
npm install
npm start
```

## Сборка дистрибутивов

- **Все платформы:** `npm run build`
- **Windows (portable exe):** `npm run build:win`
- **macOS (dmg):** `npm run build:mac`
- **Linux (AppImage):** `npm run build:linux`

Артефакты в папке `dist/`.

## Использование

1. Нажмите **«Выбрать HTML…»** и выберите один или несколько `.html` / `.htm` файлов.
2. Конвертация запускается сразу; результаты сначала сохраняются во временную папку.
3. По завершении можно выбрать папку для сохранения итоговых XLSX (**«Сохранить результаты…»**).

## Технические ограничения

- **Шрифты:** если шрифт из HTML не установлен в системе (например `MagistralC`), Excel подставит замену.
- **Размеры:** Excel и браузер используют разные модели ширин/высот; переносим их по возможности с небольшими отклонениями.

## Структура проекта

- `src/main/main.js` — процесс Electron, окно, IPC, диалоги.
- `src/main/converter.js` — парсинг HTML/CSS (формат 1С) и генерация XLSX (ExcelJS).
- `src/preload/preload.js` — preload, `contextBridge` API для рендерера.
- `src/renderer/index.html` — интерфейс: список файлов, прогресс, лог, кнопки.
